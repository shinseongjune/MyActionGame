# MyActionGame

Portfolio Unreal Engine project for a 3D TPS melee action game.

## Stack

- Unreal Engine 5.7
- Git LFS for binary assets

## Notes

- Generated folders such as `DerivedDataCache`, `Intermediate`, and `Saved` are excluded from version control.
- Unreal binary assets such as `.uasset` and `.umap` are tracked with Git LFS.
